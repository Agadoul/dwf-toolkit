This samples shows how to create every kind of resource needed in 2D package in detail. It also shows the usage of different kinds of objects. It's a configuralbe sample. User can create w2d file needed in DWF package or xaml, ttf and w2x files needed in DWFx package by modifying the classFactory value in W2DTest.in. This sample is a little different from W2DTest. There is no code to show how to process Font resource.
