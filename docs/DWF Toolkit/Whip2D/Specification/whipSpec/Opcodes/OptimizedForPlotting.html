Optimized For Plotting

Purpose

    The Optimized For Plotting opcode gives the WHIP! data generators the ability to choose whether to optimize the WHIP! data for display or plotting. This opcode categorizes WHIP! data as either optimized for plotting or for displaying.

    The viewers must honor this opcode and take full advantage of the nature of the WHIP! data while plotting or displaying the drawables from WHIP! data. When the WHIP! data is optimized for plotting, the WHIP! data generator should store enough data to allow full fidelity plots at the selected resolution and color depth. When the WHIP! data is optimized for display, allow for much deeper zooms.

Syntax
 
Opcode format Opcode Operand Format Comments
Extended ASCII (PlotOptimized <ws>|0|1 If the value is 1, then, the WHIP! data is optimized for plotting, 
Else the default  (the WHIP! data is optimized for viewing).

Details

    Only Extended  ASCII is supported,  there is no binary or extended support.
Use Case
    When the WHIP! data is "optimized for plotting" the users should send a plot-ready file to a reprographer or a colleague and be assured that they will be able to plot the right picture.
