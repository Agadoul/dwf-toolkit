Define Source Drawing Filename - DEPRECATED

Purpose

    The Define Source Drawing Filename function provides information to WHIP! data reading applications about what source file was used to generate the WHIP! data.
Syntax
 
Opcode format Opcode Operand Format Comments
Extended ASCII (SourceFilename <ws><TFilename>) Provides name of the source file that was used to create the WHIP! data
    Filename The simple filename of the source file that was used to create the WHIP! data
Details
    Some WHIP! data reading applications may need to "know" the name of the source file that was used to generate the WHIP! data. The WHIP! data generating application can indicate this need with the SourceFilename opcode:
      (SourceFilename 'House Plan.DWG')
Notes
    It is intended that the Filename argument refer only to a basic filename, and not include path information.

    WHIP! data should contain, at most, one occurrence of the source creation time opcode, which by convention should be nested inside a Define Drawing Information Block opcode.

Default
    By default, no source drawing file name is defined, which is equivalent to
      (SourceFilename unknown)
