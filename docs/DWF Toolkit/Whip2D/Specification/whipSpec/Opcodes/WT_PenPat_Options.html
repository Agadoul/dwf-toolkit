Pen Pattern Options

Purpose

The Pen Pattern Options opcode is a rendering option for storing the penset option selections in WHIP! data (as configured in the pen pattern editor)  and makes it possible to save penset options in WHIP! data . Additional parameters enable pen widths to be proportional to new plotted paper sizes; render as gray scale; and, use error diffusion instead of ordered dithering. Syntax
 
Opcode format Opcode Operand Format Comments
Extended ASCII (PenPat_Options <ws><Bscale_pen_width><Bmap_colors_to_gray_scale><ws><Buse_alternate_fill_rule>,
<Buse_error_diffusion_for_Rasters>[<ws>])		 to store penset options as configured in the pen pattern 
editor
 
    Scale Pen Width — If true, and if the WHIP! data is plotted on a different paper size than it was originally created for, then pen widths will be enlarged or reduced in proportion to the two paper sizes.

    Map Colors to gray Scale — If true, this will cause colors to be rendered as gray scale, using Buzzsaw screening patterns.

    Use Error Diffusion for Rasters — If true, error diffusion will be used instead of ordered dithering (if dithering is necessary to render any rasters embedded in the WHIP! data).
     

Notes
    The purpose is to store penset options, as configured in the pen pattern editor. The preceding parameters correspond to checkboxes on the Options page of the Properties subdialog in the Buzzsaw Pen-set-editor dialog. These options only affect the plotted output from Buzzsaw's plotting utility and they do not affect the output from Volo View.
