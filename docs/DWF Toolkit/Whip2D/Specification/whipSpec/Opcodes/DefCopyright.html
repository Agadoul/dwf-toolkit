Define Copyright - DEPRECATED

Purpose

The Define Copyright function allows a copyright notice to be added to the file that is ignored by the file reading application.

Syntax

Opcode format Opcode Operand Format Comments
Extended ASCII (Copyright <ws><TNotice>[<ws>]) A copyright notice comment

Notice A comment string representing a comment that is normally ignored by the file reading application

Details

The Copyright opcode provides the ability to add a copyright notice:

(Copyright '(c) 1998 Grokware, Inc.')

Notes

A WHIP! data reading application can provide an option to the user to view the copyright notice contained in the WHIP! data.
