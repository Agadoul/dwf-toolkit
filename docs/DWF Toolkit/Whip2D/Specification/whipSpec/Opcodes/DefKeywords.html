Define Keywords - DEPRECATED

Purpose

The Define Keywords function allows a list of keywords to be added to the file that is ignored by the file reading application. Syntax
 
Opcode format Opcode Operand Format Comments
Extended ASCII (Keywords <ws><TList>[<ws>]) A list of keywords comment
List   A comment string representing a comment that is normally ignored by the file reading application. The keywoards are separated by white space. Details The Keywords opcode provides the ability to add a list of keywords: (Keywords 'vector graphics bitmap integer') Notes WHIP! data reading applications can provide an option to the user to view the list of keywords contained in WHIP! data.
