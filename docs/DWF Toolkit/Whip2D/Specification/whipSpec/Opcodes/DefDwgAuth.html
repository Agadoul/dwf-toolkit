Define Drawing Author - DEPRECATED

Purpose

    The Define Drawing Author function specifies the person who created the WHIP! data.
Syntax
 
Opcode format Opcode Operand Format Comments
Extended ASCII (Author <ws><Tauthor>) States person who created the file
    Author A string describing who created the WHIP! data.
Details
    Some WHIP! data reading applications may need to "know" who created the WHIP! data file. The WHIP! data generating application can indicate this need with the Define Drawing Author opcode:
      (Author 'Brian P. Mathews, Autodesk, Inc.')
Notes
    WHIP! data reading applications should not expect to be able to parse the Author argument, though it can be shown to a user.

    WHIP! data should contain a maximum of one occurrence of the Define Drawing Author opcode, which by convention should be nested inside a Define Drawing Information Block opcode.

Default
    By default, no drawing author is defined:
      (Author unknown)
