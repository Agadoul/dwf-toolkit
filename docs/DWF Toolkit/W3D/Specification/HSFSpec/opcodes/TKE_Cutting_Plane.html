TKE_Cutting_Plane

Opcode
ASCII /
Hexadecimal 2F
Decimal 47

 

Operands

Variable geometry, Float a, Float b, Float c, Float d

a, b, c, d specification of cutting plane.

 

Notes

Cutting plane is defined by the standard plane equation ax+by+cz+d=0. The normal component of the plane (a,b,c) points toward the side which is removed.
