TKE_Grid

Opcode
ASCII g
Hexadecimal 67
Decimal 103

 

Operands

Byte type, Point origin, Point ref1, Point ref2, Long count1, Long count2

type indicates shape of grid
origin start point of grid
ref1, ref2 reference points
count1, count2 limits of grid extent

 

Notes

Standard HOOPS grid definition. Grid is inserted into the currently opened segment.
