TKE_User_Index

Opcode
ASCII n
Hexadecimal 6E
Decimal 110

 

Operands

Long count, (count)xLong indices, (count)xLong values

count number of index-value pairs
indices count index values
values count values associated with the corresponding indices

 

Notes

Index-value pairs for associating user-specific data with a segment.  Attribute set on the currently opened segment.
