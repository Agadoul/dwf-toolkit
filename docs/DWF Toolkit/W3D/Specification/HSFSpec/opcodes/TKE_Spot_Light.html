TKE_Spot_Light

Opcode
ASCII ^
Hexadecimal 5E
Decimal 94

 

Operands

Point position, Point target, Byte options, [Float outer], [Float inner], [Float concentration]

position location of spot light
target aiming pointof the light
options bitmask of optional settings
outer size of outermost cone of light
inner size of center "hotspot" of light
concentration falloff factor applied away from spot light center line

 

Notes

A spot light provides a number of lighting affects. It normally provides a conical area of illumination, full intensity within the inner cone, and dropping to zero at the outer cone. Concentration, if specified, is an additional falloff factor (the value is the cosine of the angle from the centerline of the cone to the point being lit (as measured from the light position), raised to the concentration power. 

For further details on what these options mean in HOOPS/3dGS please refer to the Insert_Spot_Light specification in the 3dGS Reference Manual.

Options:

01 outer cone is specified, value is degrees out from centerline
02 outer cone is specified, value is the diameter of the cone at the target
04 inner cone is specified, if bit 08 is not set, value is degrees out from centerline, otherwise value is percentage of outer cone angle
08 inner cone is specified, if bit 04 is not set, value is the diameter of the cone at the target, otherwise value is percentage of outer cone angle
10 Light position and target are relative to the camera's position & target
20 Concentration value is specified
