ASCII [

Operands

Long length, (length)xByte data, TKE_Stop_User_Data

length length of the data.
data user data

 

Notes

User_Data allows for the insertion of arbitrary data in the stream which may be skipped or used if the application understands it.

The presence of the Stop_User_Data opcode is to provide a consistency check.
