TKE_Include_Segment

Opcode
ASCII <
Hexadecimal 3C
Decimal 60

 

Operands

Byte length, (length)xByte name

length length of the segment name string
name name of segment to be included

 

Notes

The specified segment is to "included" in the current segment; it is drawn as if it were a local subsegment.  Included segment are used to instance objects within your scene graph.
