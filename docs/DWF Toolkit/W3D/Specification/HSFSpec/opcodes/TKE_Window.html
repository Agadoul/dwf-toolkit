TKE_Window

Opcode
ASCII W
Hexadecimal 57
Decimal 87

 

Operands

Variable geometry, Float left, Float right, Float bottom, Float top

left, right, bottom, top specification of window area.

 

Notes

The contents of the segment are to be drawn within a rectangular subregion of the display specified by the values, in the range -1 to 1 relative to the containing area.
