TKE_Window_Pattern

Opcode
ASCII p
Hexadecimal 70
Decimal 112

 

Operands

Byte pattern

pattern enumerated pattern type

 

Notes

Pattern used for window background areas.  A table of the different window pattern supported may be found here.

Note, in 3dGS window areas are drawn as a rectangle of primarily the window color, with a contrasting pattern overlayed. The HOOPS window pattern is usually a finer scale that the equivalent face pattern.
